# Shopping Live — Интернет-магазин

Адаптивный лендинг для телевизионного и интернет-магазина, разработанный в рамках учебной практики.

## О проекте

Проект представляет собой адаптивный веб-сайт интернет-магазина с функциональностью корзины товаров. Сайт корректно отображается на всех устройствах от мобильных телефонов до десктопных компьютеров.

## Функционал

- Адаптивный дизайн для десктопов, планшетов и мобильных устройств
- Бургер-меню для навигации в мобильной версии
- Интерактивная корзина товаров с добавлением/удалением
- Модальное окно для просмотра содержимого корзины
- Сохранение товаров в LocalStorage между сессиями
- Уведомления о действиях пользователя

## Технологии

- HTML5 для семантической разметки
- CSS3 с использованием Flexbox и Grid
- Медиазапросы для адаптивного дизайна
- JavaScript (ES6+) для интерактивности
- Local Storage API для сохранения данных

## Установка и запуск

1. Скачайте или клонируйте файлы проекта
2. Откройте файл index.html в любом современном браузере
3. Для тестирования адаптивности измените размер окна браузера

## Использование

- Нажмите кнопку "Купить" на любом товаре для добавления в корзину
- Кликните на иконку корзины в правом верхнем углу для просмотра
- В модальном окне корзины можно изменить количество или удалить товары
- Используйте бургер-меню на мобильных устройствах для навигации
